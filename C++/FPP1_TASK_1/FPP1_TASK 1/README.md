## C++ using Visual Studio

This is a project that was written in C++ using Visual Studio as my IDE. The FFP1_TASK 1.sln file is the solution file for visual studio, which includes version etc. To run the project open FFP1_TASK 1 folder and run in visual studio. Make sure to have them in the same directory. The different header files (.h) and source files (.cpp) which are used to run this project, can be found in the “FPP1_TASK 1 folder”.

The high level requirements for the console project are listed below:

“You are hired as a contractor to help a university migrate an existing student system to a new platform using C++ language. Since the application already exists, its requirements exist as well, and they are outlined in the next section. You are responsible for implementing the part of the system based on these requirements. A list of data is provided as part of these requirements. This part of the system is responsible for reading and manipulating the provided data.”
